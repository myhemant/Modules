DESCRIPTION:
------------
This module will Add a link to every article node that updates a date field to today using ajax


INSTALLATION:
-------------
1. Place the entire article directory into your Drupal sites/all/modules/
   directory.
2. Download date module and paste in site/all/modules directory  
	http://drupal.org/project/date 

3. Enable the article module by navigating to:

     administer > modules


Features:
---------
  *	Depndencies on date module
  * Date field will automatically created in "Article" content type when module will get installed.
